= Collect configuration and diagnostic information

[role="_abstract"]
To start troubleshooting {PlatformNameShort}, use the `must-gather` command on {OCPShort} or the `sos` utility on a {VMBase} to collect configuration and diagnostic information. 
You can attach the output of these utilities to your support case.
