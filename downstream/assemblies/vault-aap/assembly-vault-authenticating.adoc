= Authenticate to `hashicorp.vault`

[role="_abstract"]

After you install or migrate to the `hashicorp.vault` collection, authentication is configured in the {PlatformNameShort} user interface. An administrator creates a custom credential type to authenticate to {Vault}. Then users create credentials to use with job templates.
