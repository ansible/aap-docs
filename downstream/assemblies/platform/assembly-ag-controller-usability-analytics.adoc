= Usability Analytics and Data Collection

Usability data collection is included with {ControllerName} to collect data to better understand how {ControllerName} users interact with it.

Only users installing a trial of or a fresh installation of are opted-in for this data collection.

{ControllerNameStart} collects user data automatically to help improve the product.
