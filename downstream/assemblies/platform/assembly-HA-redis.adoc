= Caching and queueing system

[role="_abstract"]
link:https://redis.io/[Redis] provides centralized caching and queueing for {PlatformNameShort} {PlatformVers} components.
This in-memory system stores event queues, session data, and tokens to deliver improved performance, reliability, and data security.  

In {PlatformNameShort} {PlatformVers}, link:https://redis.io/[Redis (REmote DIctionary Server)] is used as the caching and queueing system.
