= Customize your {OperatorPlatformName} on {OCPShort}

[role="_abstract"]

After installing the {OperatorPlatformNameShort}, you can customize your deployment by setting configuration options for its nested components. 
You must define these parameters on the parent Automation Ansible Platform Custom Resource (CR). 
The operator automatically disseminates the configuration to each component of the platform.
