= Pull and sync images from {HubName} to your local system

:context: pulling-images-container-repository

[role="_abstract"]
Pull {ExecEnvName} from the {HubName} remote registry to make a copy to your local machine. 
{HubNameStart} provides the `podman pull` command for each `latest` {ExecEnvShort} in the container repository. 
You can copy and paste this command into your terminal, or use `podman pull` to copy an {ExecEnvShort} based on an {ExecEnvShort} tag.
