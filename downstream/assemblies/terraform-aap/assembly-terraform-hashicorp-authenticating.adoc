= Authenticating to hashicorp.terraform

[role="_abstract"]

After installing or migrating to `hashicorp.terraform`, administrators configure a custom credential type in the {PlatformNameShort} user interface to authenticate to {TerraformEnterpriseShortName} or {TerraformCloudShortName}. Then users must create credentials (based on the credential type) to use with job templates in {PlatformNameShort}.
