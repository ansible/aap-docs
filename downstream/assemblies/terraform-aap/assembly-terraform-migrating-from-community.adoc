= Migrating from {TerraformCommunityName}

[role="_abstract"]

If you want to use {PlatformNameShort} with {TerraformEnterpriseShortName} (TFE) or {TerraformCloudShortName} and you are currently using {TerraformCommunityName} (TCE), you must first migrate to TFE or {TerraformCloudShortName} and then update {PlatformNameShort} configurations to work with TFE or {TerraformCloudShortName}.
