= Building AWS AMI images for {SelfService}

[role="_abstract"]

Build raw disk images locally and convert them into Amazon Machine Images (AMI) for use on AWS.

== Prerequisites
* Ansible Core 2.15 or later
* {Python} 3.9 or later
* Podman 5.4.1 or later
* `bootc-image-builder` container image
* AWS CLI installed and configured (for the upload step only)
