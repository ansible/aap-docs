= Using custom actions and UI components in Backstage Software Templates

:context: self-service-rbac

[role="_abstract"]
The self-service automation portal is built on a {RHDH} base image. 
It uses Backstage Software Templates, which are YAML-based workflow definitions that provide user forms to execute automation tasks in Ansible Automation Platform.
