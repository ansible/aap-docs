= Create and validate an {ExecEnvShort} with the AI assistant

:context: mcp-vscode-ext-create-ee

Use the AI assistant to generate and validate a definition file, and then build an {ExecEnvShort} with `ansible-builder`.

== Prerequisites

* The GitHub Copilot Extension installed and active.
* Ansible {VSCode} extension installed.
* The Ansible MCP server is enabled. Ensure the setting `ansible.mcpServer.enabled` is set to `true`.
* {ToolsName} installed through the AI assistant (`adt_check_env`) or manually.
