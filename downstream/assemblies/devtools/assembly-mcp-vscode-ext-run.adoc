= Running playbooks with `ansible-navigator` using the AI assistant

:context: mcp-vscode-ext-run

[role="_abstract"]
Once you have built a suitable {ExecEnvShort}, you can ask the AI assistant to generate the correct ansible-navigator command to execute your playbook in that particular environment.

== Prerequisites

* The GitHub Copilot Extension installed and active.
* Ansible {VSCode} extension installed.
* The Ansible MCP server is enabled.
* {ToolsName} installed through the AI assistant (`adt_check_env`) or manually.
* You have an {ExecEnvShort} definition file.
