= Running playbooks with `ansible-navigator` using the AI assistant

:context: mcp-vscode-ext-run

[role="_abstract"]
Once you have built a suitable {ExecEnvShort}, prompt the AI assistant to generate the correct `ansible-navigator` command to execute your playbook in that particular environment.

== Prerequisites

* The GitHub Copilot Extension is installed and active.
* Ansible {VSCode} extension is installed.
* The Ansible MCP server is enabled.
* {ToolsName} is installed through the AI assistant (`adt_check_env`) or manually.
* An {ExecEnvShort} definition file is prepared and ready to use.
