= Scaffolding an Ansible project with the AI assistant

:context: mcp-vscode-ext-scaffold

[role="_abstract"]
Use the Ansible {VSCode} extension's embedded MCP server to generate a new project structure, and to create and scaffold new content such as playbooks or collections.
With the AI assistant, you can ensure the project aligns with best practices and organizational standards without manually creating boilerplate code. 

== Prerequisites

* The Ansible {VSCode} extension is installed, running, and enabled in Agent Mode.
* The Ansible MCP server is enabled in the Ansible {VSCode} extension.
* {ToolsName} is installed.
