= Configuring the Ansible {VSCode} extension MCP server

:context: mcp-vscode-ext-enable

[role="_abstract"]
As an automation developer, you can enable the Ansible MCP (Model context protocol) server to work with the Ansible {VSCode} extension. Doing so empowers {ToolsName} to give customized suggestions tuned specifically to your working environment.

You can enable the MCP server through the {VSCode} command palette, or the *Settings* menu.

== Prerequisites
* {VSCode} is installed and running.
* The GitHub Copilot extension is installed, running, and enabled in Agent Mode.
* The Ansible {VSCode} extension is installed.
