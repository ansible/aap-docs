== CVE

CSV Versions in this release:

With this update, the following CVEs have been addressed:

* link:https://access.redhat.com/security/cve/cve-2025-61140[CVE-2025-61140] `ansible-automation-platform-25/lightspeed-rhel8`: `jsonpath`: Prototype Pollution vulnerability in the value function.(AAP-64331)
* link:https://access.redhat.com/security/cve/cve-2026-24486[CVE-2026-24486] `aap-installers-rag-content`: Python-Multipart has Arbitrary File Write via Non-Default Configuration.(AAP-64174)
* link:https://access.redhat.com/security/cve/cve-2025-13465[CVE-2025-13465] `ansible-automation-platform-25/lightspeed-rhel8`: `prototype` pollution in `_.unset` and `_.omit` functions.(AAP-64097)
* link:https://access.redhat.com/security/cve/cve-2026-24049[CVE-2026-24049] `automation-controller`: `wheel`: Privilege Escalation or Arbitrary Code Execution via malicious wheel file unpacking.(AAP-63855)
* link:https://access.redhat.com/security/cve/cve-2026-24049[CVE-2026-24049] `automation-controller`: `wheel`: Privilege Escalation or Arbitrary Code Execution via malicious wheel file unpacking.(AAP-63855)

* link:https://access.redhat.com/security/cve/cve-2026-22029[CVE-2026-22029] `automation-gateway`: React Router vulnerable to XSS via Open Redirects.(AAP-62514)
