= Manage containers in {PrivateHubName}

:context: managing-containers

[role="_abstract"]
Learn the administrator workflows and processes for configuring the {PrivateHubName} remote registry and repositories.

When you set up your container repository, you must add a description, include a README, add teams that can access the repository, and tag {ExecEnvName}.

.Prerequisites

* You are logged in to Ansible Automation Platform.
* You have permissions to change the repository.
