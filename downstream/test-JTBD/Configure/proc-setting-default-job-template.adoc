= Set a default for a job template

[role="_abstract"]
To use supported execution environments, you can configure a specific execution environment globally, or you can set one per job template. Take the following steps to set an execution environment for specific automation jobs.

.Procedure

. From the Ansible Automation Platform UI, go to Automation Execution > Templates.
. Click the job template you want to configure. For example, click Demo Job Template.
. Click Edit template and specify the Execution environment you want to use. By default, this is set to Default execution environment.
. Click the execution environment name to see its details, including the location of the execution environment container image. For the default, the image resides at registry.redhat.io.
