= Cost and savings analysis metrics

[role="_abstract"]
The costs and savings analysis generates the following metrics to quantify the return on investment (ROI) derived from automation execution.

.Costs and Savings Output Metrics
[cols="1,3", options="header"]
|===
|Metric
|Description

|**Cost of manual automation**
|Total cost if all jobs were run manually

|**Cost of automated execution**
|Total cost of running jobs on {PlatformNameShort}

|**Total savings/cost avoided**
|Total cost of running jobs on {PlatformNameShort}

|**Total hours saved/avoided**
|Time saved by automation vs manual execution

|**Time taken to manually execute (min)**
|Estimated manual execution time in minutes

|**Time taken to create execution (min)**
|Estimated time spent creating or authoring the automation (e.g. writing playbooks, setting up jobs) before it could be run. Included in cost when the switch above is on.

|**Monthly cost of running {PlatformNameShort}**
|Monthly cost of running the {PlatformNameShort}. This value includes license, labor and infrastructure costs to run {PlatformNameShort}. It is used to calculate the automation savings
|===
