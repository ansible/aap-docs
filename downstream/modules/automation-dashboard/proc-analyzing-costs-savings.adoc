= Analyzing costs and savings

[role="_abstract"]
To calculate total savings, the cost and savings analysis compares manual labor costs against {PlatformNameShort} execution costs. The dashboard prorates the monthly subscription cost into daily reporting values.

.Procedure

. In the **Monthly AAP cost** field, enter the monthly cost of running the {PlatformNameShort}. This value includes license, labor, and infrastructure costs to run {PlatformNameShort}. It is used to calculate the automation savings.
. In the **Hourly rate for manually running the job ($)** field, enter the hourly labor cost used to estimate what it would cost to run these jobs manually. This is used to calculate manual cost and savings.
. Optional: Select **Time taken to create automation** to include the estimated time spent creating or authoring the automation (for example, writing playbooks, setting up jobs) before it could be run.
+
{AutomationDashboardNameStart} calculates the following data points based on your inputs:

* **Cost of manual automation**: Total cost if all jobs were run manually.
* **Cost of automated execution**: Total cost of running jobs on {PlatformNameShort}.
* **Total savings/cost avoided**: Difference between manual and automated cost.
* **Total hours saved/avoided**: Time saved by automation vs manual execution.
* **Time taken to manually execute (min)**: Estimated manual execution time in minutes.
* **Time taken to create automation (min)**: Estimated time spent creating or authoring the automation (e.g. writing playbooks, setting up jobs) before it could be run. Included in cost when the switch above is on.

.Verification

* Navigate to the Reports page to view the Total savings based on your updated cost configurations.

[NOTE]
You can export the cost and savings analysis as a CSV or PDF file. You must specify a date range to generate a PDF report.
