= Push the image to a registry
Tag and push your {ExecEnvShort} image to a container registry accessible by your Ansible Automation Platform instance:

`podman tag my-cfn-mcp-ee:latest registry.example.com/ee/my-cfn-mcp-ee:latest`

`podman push registry.example.com/ee/my-cfn-mcp-ee:latest`

If you are using the {PrivateHubName} included with {PlatformNameShort}, push to its registry.
