= Add your MCP-enabled {ExecEnvShort} to {PlatformNameShort}

[role="_abstract"]
To make your custom execution environment available for automation jobs, add it to Ansible Automation Platform by specifying the container image path and any necessary registry credentials.

.Procedure
. In the navigation panel, select {MenuACExecEnvironments}.
. Click btn:[Add].
. Enter the image path (for example, `registry.example.com/ee/my-cfn-mcp-ee:latest`).
. Configure registry credentials if required.
. Save the execution environment.
