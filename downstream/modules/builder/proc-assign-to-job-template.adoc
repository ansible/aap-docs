= Assign to a Job Template

[role="_abstract"]
To execute automation tasks using your custom tools, assign the MCP-enabled {ExecEnvShort} to a job template and configure any necessary server credentials.

.Procedure
. In the navigation panel, select {MenuAETemplates}.
. Edit or create a Job Template.
. In the *Execution Environment* field, select your MCP-enabled execution environment.
. Configure any required credentials for the MCP server (such as AWS credentials).
. Save and launch the template.
