= Validate and manage MCP servers

[role="_abstract"]
To ensure that MCP servers are correctly integrated into your {ExecEnvShort}s and function as expected, it is essential to validate their installation and manage their configurations effectively.

MCP servers are validated and managed through a unified system built into the Ansible execution environment during the image build process.
