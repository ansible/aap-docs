= Out of scope

[role="_abstract"]
Understand which {PlatformNameShort} components and configurations require manual recreation in the target environment and are not covered by the migration process.

The {TitleMigration} guide focuses on the core components of {PlatformNameShort}. The following items are currently out of scope for this migration process:

* {EDAName}: Manually recreate configuration and content for {EDAName} in the target environment.
* Instance groups: Manually recreate instance group configurations after migration.
* Hub content: Manually re-import or reconfigure content hosted in {HubName}.
* Custom Certificate Authority (CA) for receptor mesh: Manually reconfigure custom CA configurations for receptor mesh.
* Disconnected environments: The migration process does not cover disconnected environments.
* Execution environments (other than the default one): Manually rebuild or re-import custom execution environments.

Manually re-create, import, or configure these items in the target environment.
