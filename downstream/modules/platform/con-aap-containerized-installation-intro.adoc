= Containerized {PlatformNameShort} installation

[role="_abstract"]
Containerized {PlatformNameShort} uses Podman to run the platform in containers on {RHEL} host machines. 
With this installation method, you manage both the product and infrastructure lifecycle while taking advantage of containerized architecture.

Containerized {PlatformNameShort} runs as rootless containers for enhanced security by default. 
You can install and operate {PlatformNameShort} with a non-root user account.
