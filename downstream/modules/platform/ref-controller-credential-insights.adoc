= {RHLightspeed} credential type

Select this credential type to enable synchronization of cloud inventory with {RHLightspeed}.

//image:credentials-create-insights-credential.png[Credentials- create Insights credential]

{RHLightspeed} credentials are the {RHLightspeed} *Username* and *Password*, which are the user's Red Hat Customer Portal Account username and password.

The `extra_vars` and `env` injectors for {RHLightspeed} are as follows:

[literal, options="nowrap" subs="+attributes"]
----
ManagedCredentialType(
    namespace='insights',
....
....
....

injectors={
        'extra_vars': {
            "scm_username": "{{username}}",
            "scm_password": "{{password}}",
        },
        'env': {
            'INSIGHTS_USER': '{{username}}',
            'INSIGHTS_PASSWORD': '{{password}}',
        },
----
