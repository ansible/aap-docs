= Reinstalling containerized {PlatformNameShort}

[role="_abstract"]
To reinstall a containerized deployment after uninstalling and preserving the database, follow the steps in link:{URLContainerizedInstall}#installing-containerized-aap[Installing containerized {PlatformNameShort}] and include the existing secret key value in the playbook command:

----
$ ansible-playbook -i inventory ansible.containerized_installer.install -e controller_secret_key=<secret_key_value>
----
