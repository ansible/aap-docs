= Installing without internet access

[role="_abstract"]
Use the {PlatformName} *Bundle* installation program if you are unable to access the internet, or would prefer not to install separate components and dependencies from online repositories. 
Access to {RHEL} repositories is still needed. 
The `tar` archive includes all other dependencies.

.Procedure

. Navigate to the link:{PlatformDownloadUrl}[{PlatformName} download] page.
. In the *Product software* tab, click btn:[Download Now] for the *Ansible Automation Platform <latest-version> Setup Bundle*.
. Extract the files:
+
-----
$ tar xvzf ansible-automation-platform-setup-bundle-<latest-version>.tar.gz
-----
