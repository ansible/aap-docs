= Backup playbook

[role="_abstract"]
The {PlatformNameShort} installer (`install.yml`) includes the `backup.yml` playbook, which automates the backup and restoration of your environment. You can use this playbook to capture critical configuration data and databases for the following components:

* {GatewayStart}:
** The database
** The `SECRET_KEY` file
* {ControllerNameStart}:
** The database
** The `SECRET_KEY` file
** The `RESOURCE_SERVER` key
** The per-host custom configuration files
* {HubNameStart}:
** The database
** The database_fields.symmetric.key file
** The `SECRET_KEY` file
** The `RESOURCE_SERVER` key
** {HubNameStart} pulp content
* {EDAcontroller}:
** The database
** The `SECRET_KEY` file
** The `RESOURCE_SERVER` key
