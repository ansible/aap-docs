= Standalone Redis

[role="_abstract"]
Standalone Redis offers a straightforward deployment option for {PlatformNameShort} environments where high availability is not critical. 
Choose standalone Redis for development, testing, or smaller production deployments that prioritize simplicity and ease of management over redundancy.

Standalone Redis consists of a simple architecture that is easy to deploy and configure.

image::gw-single-node-redis.png[Standalone Redis deployment]

If a resilient solution is not a requirement, deploy {PlatformNameShort} with a standalone Redis.
