= Removing roles from a user

You can remove a user's roles by editing the user information in the *Roles* tab.

.Procedure

. From the navigation panel, select {MenuAMUsers}.
. Select the user name whose role access you want to remove.
. Select the *Roles* tab.
+
include::snippets/snip-gw-roles-note-multiple-components.adoc[]
+
. To remove a single role, click the *-* icon next to the role and confirm removal on the dialog that appears. 
. To remove multiple roles, select the checkbox next to each role you want to remove and click *Remove selected roles* from the *More actions {MoreActionsIcon}* list on the menu bar. On the dialog that appears, confirm removal of the selected roles and click btn:[Remove role].
