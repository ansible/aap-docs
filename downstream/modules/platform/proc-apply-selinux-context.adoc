= Apply the SELinux context

[role="_abstract"]
By applying the necessary SELinux context to the Pulp directories you ensure proper file access permissions and security policy compliance. They are essential for enabling the high availability (HA) deployment of {HubName} on SELinux.

.Procedure 

. Shut down the Pulp service:
+ 
----
$ systemctl stop pulpcore.service
----
. Unmount `/var/lib/pulp/pulpcore_static`:
+
----
$ umount /var/lib/pulp/pulpcore_static
----
. Unmount `/var/lib/pulp/`:
+
----
$ umount /var/lib/pulp/
----
. Modify the `/etc/fstab` file as follows:
+
----
srv_rhel8:/data /var/lib/pulp nfs defaults,_netdev,nosharecache,context="system_u:object_r:pulpcore_var_lib_t:s0" 0 0
----
. Run the mount command:
+
----
$ mount -a
----
