= Applying the SELinux context

[role="_abstract"]
Apply the necessary SELinux context to the Pulp directories by modifying the /etc/fstab file. This procedure is critical for ensuring proper file access permissions and security policy compliance, which are essential for enabling the high availability (HA) deployment of {HubName} on SELinux.

.Procedure 

. Shut down the Pulp service:
+ 
----
$ systemctl stop pulpcore.service
----
. Unmount `/var/lib/pulp/pulpcore_static`:
+
----
$ umount /var/lib/pulp/pulpcore_static
----
. Unmount `/var/lib/pulp/`:
+
----
$ umount /var/lib/pulp/
----
. Open `/etc/fstab` file using a text editor, then replace the existing value for `/var/lib/pulp` with the following:
+
----
srv_rhel8:/data /var/lib/pulp nfs defaults,_netdev,nosharecache,context="system_u:object_r:pulpcore_var_lib_t:s0" 0 0
----
. Run the mount command:
+
----
$ mount -a
----
