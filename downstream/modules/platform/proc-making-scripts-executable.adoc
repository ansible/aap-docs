= Making the scripts executable

[role="_abstract"]
Make sure that both scripts you created are executable by using the following procedure:

.Procedure

. Run the following command:
+
----
chmod a+x /home/my-user/cron-gather /home/my-user/cron-report
----
+
[NOTE] 
====
Make sure that you have replaced `/home/my-user/` with your actual path.
====
