= Red Hat Virtualization

[role="_abstract"]
Red Hat Virtualization (RHV) is an enterprise virtualization platform that enables organizations to create and manage virtualized environments. It provides a robust and scalable solution for running virtual machines (VMs) and managing virtualized resources.
