= Configure {ControllerName} operator route options

[role="_abstract"]

The {OperatorPlatformName} installation form provides advanced options to configure your {ControllerName} operator route.

[IMPORTANT]
====
You must assign a unique `metadata.name` to each custom resource (CR) in your namespace. If you assign an `AutomationControllerMeshIngress` the same name as your `{PlatformNameShort}` installation, the operator overrides default routes and services. This conflict causes the platform installation to fail.
====

.Procedure
. Log in to {OCP}.
. Navigate to menu:Operators[Installed Operators].
