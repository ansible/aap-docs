= Edit an authenticator

[role="_abstract"]

You can change the settings of previously configured authenticators from the *Authentication* list view. 

.Procedure

. From the navigation panel, select {MenuAMAuthentication}. 
. In the list view, you can either:
+
.. Select the btn:[Edit] image:leftpencil.png[Edit,15,15] icon next to authenticator you want to modify, or 
.. Select the authenticator name displayed in the *Name* column and click btn:[Edit authenticator] from the *Details* page.
+
. Change the authentication details or mapping configurations as required.
. Click btn:[Save].
