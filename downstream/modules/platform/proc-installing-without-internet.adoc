= Installing without internet access

[role="_abstract"]
The {PlatformName} *Bundle* installer enables you to install {PlatformNameShort} without internet access. 

.Prerequisites
* Access to {RHEL} repositories.

.Procedure
. Go to the link:{PlatformDownloadUrl}[{PlatformName} download] page.
. Click btn:[Download Now] for the *Ansible Automation Platform <latest-version> Setup Bundle*.
. Extract the files:
+
-----
$ tar xvzf ansible-automation-platform-setup-bundle-<latest-version>.tar.gz
-----
