= {NavigatorStart}

[role="_abstract"]
{NavigatorStart} is a _textual user interface_ (TUI) that becomes the primary command line interface into the automation platform.
It covers use cases from content building, running automation locally in an {ExecEnvShort}, running automation in {PlatformNameShort}, and providing the foundation for future _integrated development environments_ (IDEs).
