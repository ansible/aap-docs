= Changing your LLM model 

[role="_abstract"]

To change the LLM model for your containerized {PlatformNameShort} deployment of {AAPchatbot}, you must edit the inventory file with the specific details of your new LLM provider and then rerun the install playbook.

.Procedure

. Edit the inventory file to update the following {AAPchatbot} variables with the specific details of your desired LLM provider:

* `lightspeed_chatbot_model_url`
* `lightspeed_chatbot_model_api_key`
* `lightspeed_chatbot_model_id`
* `lightspeed_chatbot_default_provider`

. Rerun the `install` playbook to xref:installing-containerized-aap[install the containerized {PlatformNameShort}].
