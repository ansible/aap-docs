= Centralized Redis

[role="_abstract"]
Redis provides fast, in-memory caching and session management for {PlatformNameShort} components, improving performance and enabling scaling.

{PlatformNameShort} offers a centralized Redis instance in both xref:gw-single-node-redis_planning[standalone] and xref:gw-clustered-redis_planning[clustered] topologies.
