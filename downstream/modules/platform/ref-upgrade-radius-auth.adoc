= Authentication type: RADIUS

[role="_abstract"]
Review the general settings and mappings for RADIUS authentication. 
Compare how configurations transform from {ControllerName} 2.4 to the {Gateway} {PlatformVers}.

*General settings*

[cols="50%,50%",options="header"]
|====
| {ControllerNameStart} 2.4 | {GatewayStart} 2.6
a| ----
RADIUS_SERVER: "radius.example.com"
RADIUS_PORT: 1812
RADIUS_SECRET: "shared-secret"
---- a| ----
"configuration": {
  "SERVER": "radius.example.com",
  "PORT": 1812,
  "SECRET": "shared-secret"
}
----
|====

*Mappings*

RADIUS authentication does not support user mappings in either {ControllerName} 2.4 or {GatewayStart} {PlatformVers}.
