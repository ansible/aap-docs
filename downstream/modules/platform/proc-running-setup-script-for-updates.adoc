= Running the {PlatformName} installer setup script

[role="_abstract"]
Execute the {PlatformName} installer setup script after configuring your `inventory` file. This action initiates the installation or update process, applying all your custom settings to deploy the platform successfully.

.Procedure

* Run the `setup.sh` script:
+
-----
$ ./setup.sh
-----
+
The installation will begin.
