= CyberArk Central Credential Provider Lookup credential type

[role="_abstract"]
The CyberArk Central Credential Provider (CCP) Lookup credential type allows {ControllerNameStart} to retrieve credentials from a CyberArk vault by using the Central Credential Provider REST API.

This is considered part of the secret management capability. 

For more information, see link:{URLControllerAdminGuide}/assembly-controller-secret-management#ref-cyberark-ccp-lookup[CyberArk Central Credential Provider (CCP) Lookup].
