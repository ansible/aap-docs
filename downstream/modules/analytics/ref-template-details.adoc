= Top template details and metrics

[role="_abstract"]
The following table provides definitions for the data points associated with each template listed in the *Top templates* view of the Automation Calculator.
This information is used to calculate automation savings.

.Top Template Metrics
[cols="1,3", options="header"]
|===
|Metric
|Description

|Total elapsed sum
|Total run time across all runs of the template.

|Success elapsed sum
|Total run time specifically for successful template runs.

|Failed elapsed sum
|Total run time specifically for failed template runs.

|Automation percentage
|The percentage of total organizational automation that this specific template accounts for.

|Associated organizations
|The organizations that this template runs against.

|Associated clusters
|The {ControllerName} clusters that the template runs on.
|===
