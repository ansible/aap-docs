= What users see

[role="_abstract"]
Auto-generated templates appear in the template catalog alongside custom templates.
Users browse or search the catalog and select a template to run.

When a user opens an auto-generated template, {SelfServiceShort} displays a form.
The form fields come from the {PlatformNameShort} Job Template's Survey questions and "Prompt on Launch" options.
Users select {PlatformNameShort} resources by name (such as Inventories or Credentials) and answer survey questions (such as application name or environment).
Authentication is handled automatically and is not visible on the form.

After the user submits the form, {SelfServiceShort} launches the {PlatformNameShort} Job Template and displays an output page.
The output page shows the job ID, job status, and optional links such as a direct link to the job in {PlatformNameShort}.

Users do not need to understand YAML or the template structure.
The form and output page handle everything.
