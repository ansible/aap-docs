= Log into OpenShift CLI

[role="_abstract"]
To deploy a plug-in registry, you must install the OpenShift CLI (`oc`) locally and log in to your cluster.

See link:{BaseURL}/openshift_container_platform/4.8/html/cli_tools/openshift-cli-oc#installing-openshift-cli[Installing OpenShift CLI in the OpenShift Container Platform] documentation.

.Procedure

. Log in to {OCPShort} using your credentials:
+
----
oc login <OpenShift_API_URL> -u <username>
----
. Create a new project or switch to an existing project:
+
----
oc new-project <project-name>
----
+
Or:
+
----
oc project <project-name>
----
