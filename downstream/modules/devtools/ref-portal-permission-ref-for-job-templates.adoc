= Permissions reference for {PlatformNameShort} job template access

[role="_abstract"]
Permissions for {PlatformNameShort} job templates

[cols="25%,20%,15%,40%",options="header"]
|===
|Permission |Resource type |Policy |Description

|`catalog.entity.read`
|catalog-entity
|read
|Users can view synchronized {PlatformNameShort} job templates in the {SelfService}.

|`scaffolder.template.parameter.read`
|scaffolder-template
|read
|Users can read template parameters.

|`scaffolder.action.execute`
|scaffolder-action
|use
|Users can execute actions through templates.

|`scaffolder.task.create`
|
|create
|Users can trigger the execution of {PlatformNameShort} job templates.

|`scaffolder.task.read`
|
|read
|Users can view task execution history and logs on the **History** page.

|`scaffolder.task.cancel`
|
|use
|Users can cancel currently running templates.
|===
