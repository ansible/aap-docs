= Configuring the output section

[role="_abstract"]
The `output` section defines what users see after a template runs.
You can display text messages, reference user input from `parameters`, show job execution data from `steps`, and provide external links.

The `output` section supports two types of content:

* *`text`:* Displays text blocks with titles and markdown content.
* *`links`:* Displays clickable buttons that open external URLs.
