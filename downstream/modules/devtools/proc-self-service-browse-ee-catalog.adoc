= Browsing the execution environment catalog

[role="_abstract"]
Browse the execution environment catalog to find and manage EE definitions in {SelfServiceShort}.
The catalog displays execution environment definitions discovered from your synchronized repositories, allowing you to view their configurations and access their files.
Use search and filtering to locate specific execution environments.

.Prerequisites

* You have logged in to {SelfServiceShort}.
* Repositories containing execution environment definitions have been synced to the portal.

.Procedure

. Navigate to the execution environment catalog.
+
[NOTE]
====
Navigation path and catalog interface details will be documented here with screenshots.
====
+
. Browse or search for execution environment definitions.
+
[NOTE]
====
Search, filter, and sort options will be documented here with screenshots.
====
+
. View detailed information about an execution environment definition.
+
[NOTE]
====
Detail view contents and available options will be documented here with screenshots.
====
+
. To download an EE definition, click *Download EE files*.
+
The EE definition files download as a `.tar` archive.

.Verification

. Verify you can locate your expected execution environment definitions in the catalog.
. If expected EEs are missing, verify the source repository has been synced successfully and contains valid Backstage catalog entity files.
