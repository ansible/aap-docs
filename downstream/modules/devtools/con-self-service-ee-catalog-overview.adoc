= The {ExecEnvShort} catalog

[role="_abstract"]
The {ExecEnvShort} (EE) catalog in {SelfServiceShort} provides a centralized view of all {ExecEnvShort} definitions created through the EE Builder.
Use the catalog to browse, search, and access EE definitions for your automation workflows.

== Catalog content

The EE catalog displays {ExecEnvShort} definitions that have been created through the EE Builder.

== Using the catalog

The catalog allows you to:

* Browse available {ExecEnvShort} definitions
* View EE configurations and associated files
* Download EE definition files
