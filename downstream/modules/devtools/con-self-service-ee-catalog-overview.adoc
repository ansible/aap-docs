= The {ExecEnvShort} catalog

[role="_abstract"]
The {ExecEnvShort} (EE) catalog in {SelfServiceShort} provides a centralized view of all {ExecEnvShort} definitions discovered from your synchronized Git repositories.
Use the catalog to browse, search, and access EE definitions for your automation workflows.

== Catalog content

The EE catalog displays {ExecEnvShort} definitions that have been created through the EE Builder or discovered from Git repositories during synchronization.

When {SelfServiceShort} syncs a Git repository, it automatically discovers and catalogs {ExecEnvShort} definitions based on Backstage entity files.

== Using the catalog

The catalog allows you to:

* Browse available {ExecEnvShort} definitions
* View EE configurations and associated files
* Download EE definition files
