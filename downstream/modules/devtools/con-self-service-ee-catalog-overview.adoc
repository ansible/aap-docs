= Understanding the execution environment catalog

[role="_abstract"]
The execution environment (EE) catalog in {SelfServiceShort} provides a centralized view of all execution environment definitions discovered from your synchronized Git repositories.
Use the catalog to browse, search, and access EE definitions for your automation workflows.

== Catalog content

The EE catalog displays execution environment definitions that have been created through the EE Builder or discovered from Git repositories during synchronization.

When {SelfServiceShort} syncs a Git repository, it automatically discovers and catalogs execution environment definitions based on Backstage entity files.

== Using the catalog

The catalog allows you to:

* Browse available execution environment definitions
* View EE configurations and associated files
* Download EE definition files
