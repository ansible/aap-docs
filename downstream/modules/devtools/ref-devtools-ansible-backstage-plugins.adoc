= Ansible backstage plugins

[role="_abstract"]
The portal's capabilities are delivered through *Ansible Backstage Plugins* that extend {RHDH} functionality; the base {RHDH} image is not customized. 
These plugins provide additional Backstage actions and filters that you use to create custom software templates.

|===
|Plugin |Functionality

|auth-backend-module-rhaap-provider
|Provides OAuth 2.0 authentication with {PlatformNameShort}.

|catalog-backend-module-rhaap
|Synchronizes {PlatformNameShort} job templates as Backstage Software Templates.

|scaffolder-backend-module-backstage-rhaap
|Provides the `rhaap:launch-job-template` action.

|backstage-rhaap-common
|Contains shared libraries and utilities for {PlatformNameShort} integration.

|self-service
|Provides the user interface for all listed functionality.
|===
