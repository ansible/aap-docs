= Disable the MCP server

[role="_abstract"]
Take the following steps to disable the MCP server in {VSCode}.

.Procedure
* To disable the server from the {VSCode} command palette, find and select *Disable {ToolsName} MCP Server*.
* To disable the server from the Ansible {VSCode} settings menu, navigate to *Extensions* > *Ansible* > *MCP Server*, and un-check the checkbox next to *Enable the Ansible MCP server for AI integration*.
