= Configure plug-in delivery for disconnected environments

[role="_abstract"]
Configure how {SelfServiceShort} retrieves plug-ins in your disconnected environment. This configuration ensures that the portal can access the required plug-ins from your mirrored registry or local plug-in registry without requiring internet connectivity.

.Procedure

* *For OCI delivery:* Create the `<release-name>-dynamic-plugins-registry-auth` secret with credentials for your disconnected registry that hosts the mirrored Ansible plug-ins OCI artifacts. See xref:oci-container-delivery_self-service-create-ocp-registry[OCI container delivery].
* *For HTTP plug-in registry:* Deploy the plug-in registry in your disconnected OpenShift project. See xref:http-plug-in-registry_self-service-create-ocp-registry[HTTP plug-in registry].
