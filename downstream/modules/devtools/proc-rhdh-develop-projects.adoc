= Developing projects

[role="_abstract"]
{DevSpacesShort} provides a web-based Integrated Development Environment that includes the Ansible {VSCode} extension and command line tools. 

.Procedure
* Develop projects in {DevSpacesShort} using the {PlatformNameShort} plug-ins for {RHDH}.
+
They provide:
+
** Links for editing projects
** Default environment with the Ansible {VSCode} extension (supporting {LightspeedShortName})
** Command line tools for executing automation tasks
