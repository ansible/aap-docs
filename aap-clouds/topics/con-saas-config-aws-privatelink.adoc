= Configure {AWSPrivateLink}

[role="_abstract"]
{AWSPrivateLink} establishes secure connectivity between your VPC and the Red Hat managed control plane without traversing the public internet.

[NOTE]
====
You must create *two* support tickets for bi-directional connectivity:  

* One enabling {AWSPrivateLink} connectivity *into the* control plane
* One enabling {AWSPrivateLink} connectivity *out of the* control plane. 
====
