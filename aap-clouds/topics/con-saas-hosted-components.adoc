= Hosted components

[role="_abstract"]
The objective of this offering is to provide an {PlatformNameShort} deployment as a managed service, relieving customers of managing the {PlatformNameShort} control plane. 

All {PlatformNameShort} capabilities in the operator-based deployment model are supported.
