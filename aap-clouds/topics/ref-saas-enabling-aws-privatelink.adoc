= Enable {AWSPrivateLink} connectivity

[role="_abstract"]
To enable private link connectivity, submit a customer support ticket and the Red{nbsp}Hat team will work with you on the next steps. 

[NOTE]
====
You must create *two* seperate support tickets for bi-directional connectivity:  

* One enabling {AWSPrivateLink} connectivity from Customer VPC *into the* control plane
* One enabling {AWSPrivateLink} connectivity *out of the* control plane to Customer VPC. 
====
