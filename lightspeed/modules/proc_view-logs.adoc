= Access and view the audit logs

[role="_abstract"]
The Ansible Visual Studio (VS Code) extension now records all {LightspeedShortName} operations in an audit log for future use. Each interaction is recorded with a timestamp, the type of action performed, details of the requested task, and other relevant information. 

The logs are displayed in the Ansible Lightspeed Output Channel of the VS Code editor and are available until you close VS Code.

.Procedure

. Open VS Code.
. Open the Command Palette of the VS Code editor.
. Click menu:Output[Show Output Channels], and then select *Ansible Lightspeed*. 
+
An *Output* panel is displayed at the bottom of the VS Code editor with a log of all user actions.
