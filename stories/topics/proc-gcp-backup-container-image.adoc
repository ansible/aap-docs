= Pulling the ansible-on-clouds-ops 2.3 container image

.Procedure
* Pull the docker image for the `ansible-on-clouds-ops` 2.3 container with the same tag as the foundation deployment.
+
[source,bash]
----
$ export IMAGE=registry.redhat.io/ansible-on-clouds/ansible-on-clouds-ops-rhel8:2.3.20230211
$ docker pull $IMAGE
----

For EMEA regions (Europe, Middle East, Africa) run the following command instead:

[source, bash]
----
$ export IMAGE=registry.redhat.io/ansible-on-clouds/ansible-on-clouds-ops-emea-rhel8:2.3.20230211
$ docker pull $IMAGE
----
