= Pulling the ansible-on-clouds-ops 2.2 container image

.Procedure
* Pull the `ansible-on-clouds-ops 2.2` container image with the same tag version as the foundation deployment.

[source,bash]
----
$ export IMAGE=registry.redhat.io/ansible-on-clouds/ansible-on-clouds-ops-rhel8:2.2.20230211
$ docker pull $IMAGE
----

For EMEA regions (Europe, Middle East, Africa) run the following command instead:

[source, bash]
----
$ export IMAGE=registry.redhat.io/ansible-on-clouds/ansible-on-clouds-ops-emea-rhel8:2.2.20230211
$ docker pull $IMAGE
----
